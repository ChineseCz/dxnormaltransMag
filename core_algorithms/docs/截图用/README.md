# Autoencoder Demo (kk.py)

`kk.py` contains a lightweight NumPy-based linear autoencoder that loads the digits dataset, trains the encoder/decoder pair, and prints latent representations plus reconstruction errors.

## Quick start

```bash
python core_algorithms/截图用/kk.py
```

The script will train for 30 epochs and report training/validation loss for each epoch.

## Dependencies

- `numpy`
- `scikit-learn`
