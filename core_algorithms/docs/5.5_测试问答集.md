3. 给出严重程度判断（如：需停机检测）
4. 给出后续处置方向（如：直流电阻测量 / FRA频率响应分析）

**检验意图：** 考察KG是否能从"现象(磁通密度偏低)→根因(匝间短路)"完成图谱路径检索
